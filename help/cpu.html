Interactive

This CPU simulator is an implementation of the data section of the Pep/8 CPU as shown in Figure 12.2 of the textbook Computer Systems, J. Stanley Warford, Fourth edition, Jones and Bartlett, Publishers, 2010.

The simulator has two modes of operation — interactive and microcode. This help page describes the interactive mode of operation. See the Microcode help page for instructions on the microcode mode of operation.

Reference: Computer Systems, Chapter 12, Section 12.1.

With the interactive mode, the user sets the control signals individually for a single cycle. There are six kinds of control signals — register address signals, multiplexer signals, ALU function signals, the ANDZ signal, the memory read/write signals, and the clock signals.

Register address signals

There are three register address signals — A, B, and C. Each signal is five bits and accesses one of the 32 one-byte registers in the two-port register bank. To set a register address signal, enter a decimal integer value in the range 0..31.
