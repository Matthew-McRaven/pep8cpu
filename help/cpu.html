CPU

This CPU simulator is an implementation of the data section of the Pep/8 CPU as shown in Figure 12.2 of the textbook Computer Systems, J. Stanley Warford, Fourth edition, Jones and Bartlett, Publishers, 2010.

The simulator has two modes of operation — single-cycle and microcode. This help page describes the single-cycle mode of operation. See the Microcode help page for instructions on the microcode mode of operation.
